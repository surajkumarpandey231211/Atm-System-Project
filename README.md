# ATM Management System

A simple ATM Management System built using Java. This project demonstrates core Java concepts such as Classes, Objects, Methods, Encapsulation, Conditional Statements, and Loops.

## Features
- Check Account Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu-Driven Console Interface

## Technologies Used
- Java
- OOP (Object-Oriented Programming)
- Scanner Class

## Learning Outcomes
- Classes and Objects
- Encapsulation
- Method Creation
- User Input Handling
- Conditional Logic
- Console-Based Application Development

This project is suitable for beginners who want to practice Core Java and OOP concepts.
